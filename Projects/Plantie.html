Plant Simulator
Hobby project (last update: 12.01.2021)
Simulator of owning a plant, main goal of the game is to choose a plant and make it grow as long as possible. Player can water the plant, fertilize it and defend from bugs that appear during gameplay. For each level of plant, Player gets coins which can be exchanged for rewards in shop implemented in game.
Technology
Unity 2019.4.8f1, C#
Developed features
Implemented mechanics such as watering the plant, fertilizing it and defending from bugs
Implemented mechanic for water and fertilizer levels and punishing the player when done incorectly
Implemented shop with ability to exchange coins for rewards such as different pot, better fertilizer or bug spray
Implemented bugs that attack randoml and destroy plant
Implemented inventory system for rewards bought from store
