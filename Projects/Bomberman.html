Bomberman
Hobby project (last update: 16.03.2019)
Traditional game of Bomberman without the use of PowerUps. Each player has 3 Hearts (HP) and has an ability to put down bombs. Bombs damage Enemies but also can destroy boxes and hurt player. Game ends whenever one of the players does not have any hearts anymore.
Technology
Allegro 5, C++
Developed features
Implemented algorith A* for AI path finding
Implemented custom UI for sound settings
Implemented scoring system for players
Implemented ability to play vs another local player or AI enemy
Implemented ability to change skin for second player or AI enemy
