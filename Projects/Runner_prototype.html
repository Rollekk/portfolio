Parkour prototype
Hobby project (last update: 16.10.2021)
Project started as a player controller for parkour movement. Goal was to make the controller as flexible as possible to use it for future projects. Added elements where basic movement as well as parkour elements like wall running.
Technology
Unity 2018.4.20f1, C#
Developed features
Impleneted players basic locomotion
Implemented parkour movement set: double jump, wall running, running, sliding
Implemented slopes to block player from moving on certain angle
Implemented ability to fight and deal damage to enemies
Implemented pick up items
Implemented climbing system
Implemented momentum system for player, rewarding him for keeping movement up
