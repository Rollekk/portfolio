Stealth game
Hobby project (last update: 09.03.2021)
Stealth game, main goal of the game is to move through map without being detected by enemy. Core mechanic is Player being vulnerable to light, whenever he is in lit environment, he gets damaged but can heal while in shadows.
Technology
Unreal Engine 4.22, C++, Blueprints
Developed features
Implemented player basic movement, parkour elements, mechanic with light and dark
Implemented melee combat and ability to throw shurikens
Implemented AI characters, their behaviour for patroling and attacking player when spotted
Implemented AI patroling system whenever in idle state
Implemented saving and loading system
Implemented two maps, tutorial and first level
Implemented mechanics for hidnig in objects to avoid being detected
Implemented lamps that can be turned off by shuriken
