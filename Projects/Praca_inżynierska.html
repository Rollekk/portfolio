EscapeRoom
Engineer's Thesis (last update: 21.10.2019)
Game made for my Engineer's Thesis. It simulates real Escape Rooms that can be experienced in real world. Player needs to figure out all puzzles inside room to get out and progress further into the game. There are two rooms avilable and one short tutorial.
Technology
  • Unreal Engine 4.22, C++, Blueprints
Developed features
  • Implemented players baisc movement and interactions with world.
  • Created and implemented all room puzzles
  • Implemented all interactable items (ex. keys) and equipable items (ex. flashlight, backpack)
  • Implemented feature for inspecting items, used for looking at them from every angle
  • Implemented saving and loading system
  • Implemented UI interface and settings panel
  • Implemented inventory mechanic for storing and using items inside backpack
