Stealth game
Hobby project (last update: 07.09.2021)
Project which was an attempt to create FPS game with puzzle elements where main mechanic is bouncing projectile. Bullet shot from gun has a trail that is different with each powerup, it can kill enemies or power electric devices.
Technology
Unity 2019.4.8f1, C#
Developed features
Implemented basic movement system for player
Implemented weapon pickup to choose from variety of options
Implemented bullet that bounces off walls and comes back to player after given time
Implemented dummy enemies to test out damage system
Implemented boxes that absorb bullet trail and get buffed from it (for example to power light into room)
